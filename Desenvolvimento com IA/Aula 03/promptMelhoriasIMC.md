Para melhorar um **programa simples de IMC**, podemos adicionar funcionalidades que o deixem **mais útil, mais robusto e mais próximo de um sistema real**. Algumas melhorias possíveis:

* validação de entrada (evitar valores negativos ou texto)
* permitir cálculo para várias pessoas
* mostrar tabela de classificação
* calcular peso ideal aproximado
* organizar o código em funções
* permitir repetição do cálculo
* melhorar a apresentação do resultado


---

# Prompt CTIF – Sistema de Cálculo de IMC Melhorado

## C – Contexto
   * Você é um programador experiente em python especializado em informática médica e está desenvolvendo um sistemade IMC.

---

## T – Tarefa
	* Alterar o código anexo para incluir novas funcionalidades ao sistema.


---

## I – Instruções
  * Criar validações para as entradas do sistema. Não permitir números negativos e não permitir alturas superiores a 2.70 metros.
	* O sistema deverá permitir o cálculo para várias pessoas sem sair do sistema.
	* O sistema deverá calcular o pesos ideal aproximado

---

## F – Formato da resposta

A resposta deve conter:

1. Uma explicação breve do funcionamento do programa.
2. O código completo em Python.
3. Comentários no código explicando a lógica utilizada.
4. Exemplo de execução do programa no terminal.
